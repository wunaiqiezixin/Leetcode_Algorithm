# P0310:汉诺塔问题
[题目链接](http://cxsjsxmooc.openjudge.cn/dspythonbookqh2/P0310/)  
## Recursion
- 递归就是函数调用自身
- 递归离不开函数，因此根据实际问题定义一个函数是十分重要的
- 早期有的编程语言只有递归，没有循环
##### 递归的作用
- 解决本来就是用递归定义的问题--斐波那契数列、绘制n阶雪花曲线
- 用递归代替多重循环--N皇后问题
- 将问题分解为规模更小的子问题进行求解(分治思想)
---
### 本题思路：
- 每次只能移动一个盘子
- 要将A地中按照大的在下小的在上规则摆放的n个盘子（不可以让任何大的盘子摆在比其小的盘子上）移动到C地，用B地作为中转
- 根据问题定义函数:
```C
void Hanoi(int n, char src,char mid,char dest);
//表示要将n个盘子，从src地，以mid地作为中转，移动到dest地
```
- 现在我们根据Hanoi来拆解问题
1. 首先，将A地摆在上面的n-1个盘子，以C地作为中转，移动到B地
```C
Hanoi(n - 1,'A','C','B')
```
2. 其次，将A地剩下的那个盘子，以B地作为中转，移动到C地
```C
Hanoi(1,'A','B','C')
```
3. 最后，将B地的n-1个盘子，以A地为中转，移动到C地
```C
Hanoi(n - 1,'B','A','C')
```
### 完整代码
### ***C***
```C
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>

void Hanoi(int n, char src, char mid, char dest);

int main(void)
{
	int n;
	scanf("%d", &n);
	Hanoi(n, 'A', 'B', 'C');
}

void Hanoi(int n, char src, char mid, char dest)
{
	if (n == 1)
	{
		printf("%c->%c\n", src, dest);
		return;
	}
	//先将a中n-1个盘子，从a->b
	Hanoi(n - 1, src, dest, mid);
	//再将a剩下的一个盘子，从a->c
	Hanoi(1, src, mid, dest);
	//最后将b中n-1个盘子从b->c
	Hanoi(n - 1, mid, src, dest);
}
```
