# P0250:波兰表达式
[题目链接](http://cxsjsxmooc.openjudge.cn/dspythonbookqh2/P0250/)

### 用递归解决
#### 波兰表达式的递归定义：
- 一个数是一个"波兰表达式" ,其值为波兰表达式的值
- 一个 "波兰表达式" 为："运算符"  "波兰表达式"  "波兰表达式"

 ---
 - 定义函数Notation(),表示输入一个波兰表达式，输出其值
```C
double Notation(void);
```
- main()函数
```C
int main(void)
{
    printf("%f\n",Notation());
}
```
- 主要任务，完成Notation函数
- 定义一个字符数组s，用来存放输入。我们用scanf()进行输入，即遇到空格时，表达式会被截断存入字符数组s。而空格后的部分会被scanf()放入一个地方，等待下一次scanf()的调用。
```C
double Notation(void)
{
    char s[100]; //给足够的大小
    scanf("%s",s);
```
- 关注s[0],若s[0]为加号，说明，要将其后的两个"波兰表达式"的值相加
```C
return Notation() + Notation();
```
- 减号、乘号、除号同理
```C
return Notation() - Notation();
return Notation() * Notation();
return Notation() / Notation();
```
- 若为数字，则直接返回字符串s表示的数字的值，用<math.h>中的atof(str) , 可以将字符串str转换成double类型
```C
return atof(s);
```
- 完整的Notation()函数
```C
double Notation(void)
{
	char s[100];
	scanf("%s", s);
	switch (s[0])
	{
	case '+': return Notation() + Notation(); 
	case '-': return Notation() - Notation(); 
	case '*': return Notation() * Notation(); 
	case '/': return Notation() / Notation(); 
	default: return atof(s);
		break;
	}
}
```
