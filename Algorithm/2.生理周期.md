# 2.生理周期
[题目链接](http://cxsjsxmooc.openjudge.cn/dspythonbookqh2/P0150/)

---
### 常见想法
k在范围[d+1,21252]内逐一枚举  
验证：(k-p)%23 && (k-e)%28 && (k-i)%33 是否成立

---
### 剪枝
从d+1天开始，先1天1天的试找出第一次体力高峰  
因为体力高峰的周期是23天，所以再23天23天的试找出体力高峰和情感高峰出现的同一天  
又因为情感高峰的周期是28天，所以再23*28天的试直到找出体力、情感、智力高峰出现的同一天

---

### ***C***

```C
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>

int main(void)
{
	int p, e, i, d;
	scanf("%d %d %d %d", &p, &e, &i, &d);
	/*求体力高峰，1天1天的枚举*/
	int k;
	for (k = d + 1; (k - p) % 23; k++);
	/*求体力、情感高峰，23天23天的枚举*/
	for (; (k - e) % 28; k += 23);
	/*求体力、情感、智力高峰，23*28天23*28的枚举*/
	for (; (k - i) % 33; k += 23 * 28);

	printf("%d", k - d);
}
```
