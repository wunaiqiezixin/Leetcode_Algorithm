# 多多的登山挑战
- [题目链接](https://kamacoder.com/problempage.php?pid=1472)

---

## 题目核心理解
这个问题的关键在于正确理解两个核心规则：
- 补给只能增加不能减少：离开营地时只能从营地拿取新的补给包，不能放下已携带的，因此携带量是一个非递减序列。
- 抵达即补满：每次到达一个营地时，携带的补给包会被补满到离开上一个营地时的数量。
  
这意味着，走一条边需要的补给包数量不能超过当前的携带量。我们的目标是找到一条从 1 到 n 的路径，使得最终携带的补给包数量最少。  

## 解法思路：动态规划 + 二分答案

### 核心洞察
最终携带的补给包数量，一定等于路径上所有边权的最大值。因为我们只会在遇到需要更大携带量的边时，才会在之前的营地补充补给到刚好满足要求的数量，以最小化最终携带量。  

因此，问题转化为：找到最小的 K，使得存在一条从 1 到 n 的路径，满足：  
1. 路径上所有的边的权值小于等于K  
2. 我们能通过在路径上的营地获取补给，使得携带量能够满足所有边的要求  


### 二分答案

- 左边界: 1 (因为边的权值 >= 1)
- 右边界: max_w (边的最大权值)  

### 动态规划验证可行性  

对于每个候选的 K，我们使用动态规划来验证是否存在满足条件的路径：  

- 定义S[u]：离开营地 u 时能携带的最大补给包数量（取最大值是为了尽可能满足后续边的要求）
- 初始化：S[1] = A[1]（离开营地 1 时最多拿 A1 个补给），其余为 - 1 表示不可达
- 转移：按营地编号递增的顺序（天然拓扑序，因为所有边 u→v 满足 u<v）遍历每个营地 u。
对于每条边 u→v，如果边权 w≤K 且S[u]≥w（当前携带量足够走这条边），则更新S[v] = max(S[v], S[u] + A[v])（到达 v 后可以再拿 A[v] 个补给）
- 验证：如果S[n]≠-1，说明存在到达终点的路径，K 是可行的

### 复杂度分析

- **时间复杂度：** O ((n+m) log (max_w))。二分查找最多进行约 30 次（max_w ≤ 1e9），每次验证遍历所有节点和边，时间复杂度 O (n+m)
- **空间复杂度：** O (n+m)。主要用于存储邻接表和 S 数组

## 代码实现
### ***C++***

```CPP
#include <iostream>
#include <vector>
#include <cstdint>
using namespace std;

//边结构体
struct Edge {
    int v;
    int64_t w;//边的权值用64为有符号整数存
};

int main() {
    //输入输出优化
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;
    vector<int64_t> A(n + 1);
    for (int i = 1; i <= n;  ++i) cin >> A[i];
    vector<vector<Edge>> adj(n + 1);//有向无环图(拓扑序)
    int64_t max_w = -1;//边的最大权值
    while (m --) {
        int u, v;
        int64_t w;
        cin >> u >> v >> w;
        adj[u].push_back({v, w});
        if (max_w < w) max_w = w;//记录最大权值(二分的上界)
    }

    //动态规划验证K的可行性
    auto check = [&](int64_t K) {
        //S[u]:离开u时可以携带的最大补给包数量
        vector<int64_t> S(n + 1, -1);
        S[1] = A[1];
        //状态转移
        for (int u = 1; u < n; ++u) {
            if (S[u] == -1) 
                continue;
            for (const auto& edge : adj[u]) {
                if (edge.w <= K && S[u] >= edge.w) {
                    S[edge.v] = max(S[edge.v], S[u] + A[edge.v]);
                }
            }
        }
        return S[n] != -1;//是否可以到达
    };

    //二分查找
    int64_t ans = -1;//找不到返回-1
    int64_t low = 1, high = max_w;
    while (low <= high) { //左闭右闭
        int64_t mid = low + (high - low) / 2;
        if (check(mid)) {
            ans = mid;
            high = mid - 1;//找到了，从更小的区间找更小的
        } else {
            low = mid + 1;//找不到，从大区间找
        }
    }
    cout << ans << '\n';
    return 0;
}
```
