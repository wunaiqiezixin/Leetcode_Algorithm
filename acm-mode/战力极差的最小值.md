# 战力极差的最小值

[题目链接](https://kamacoder.com/problempage.php?pid=1477)

---

## 一、题目深度解析
**核心问题**  
>给定 n 支战队，每支战队有 m 名成员，每名成员有一个战力值。要求从每支战队中恰好选出 1 名成员，组成一个 n 人团队，使得团队中战力值的极差（最大值 - 最小值）最小。  

**数据范围与难点**  
- 1 ≤ n ≤ 3000，1 ≤ m ≤ 100，总成员数 n*m ≤ 3e5
- 暴力枚举所有可能的组队方案：时间复杂度为 O(m^n)，完全不可行
- 核心挑战：如何高效地找到满足 "每支战队至少 1 人" 条件的最小极差区间  


## 二、核心思路：问题转化 + 滑动窗口
**关键问题转化**
本题可以等价转化为滑动窗口的经典问题：  
>将所有成员的战力值按从小到大排序后，寻找一个最短的连续区间，使得该区间内至少包含来自每支战队的 1 名成员。这个区间的长度（右端点战力值 - 左端点战力值）就是我们要找的最小极差。  

**为什么这个转化是正确的？**  
- 区间的左右端点天然对应了该区间内所有元素的最小值和最大值  
- 只要区间包含了每支战队至少 1 名成员，我们就可以从每支战队中选出 1 名在区间内的成员，此时他们的极差一定 ≤ 区间的极差  
- 我们要找的最小极差，必然对应着某个满足条件的最短区间   

**算法步骤**  
1. 扁平化与排序：将所有成员的战力值与其所属战队编号打包成一个结构体，然后按战力值从小到大排序  
2. 滑动窗口初始化：  
左指针 `left = 0`，右指针 `right = 0`  
频次数组 `team_count[n]`：记录当前窗口内每个战队的成员数量  
计数器 `count`：记录当前窗口内包含的不同战队数量  
全局最小极差 `min_diff = INT_MAX`  
3. 扩展右边界：不断向右移动 `right` 指针，将成员加入窗口  
如果该成员所属战队是第一次进入窗口，`count++`  
4. 收缩左边界：当 `count == n`（窗口已包含所有战队）时：  
计算当前窗口的极差，更新全局最小值  
尝试向右移动 `left` 指针缩小窗口  
如果移出的成员是其战队在窗口内的最后一个，`count--`  
5. 返回结果：遍历结束后，`min_diff` 即为答案  



## 三、复杂度分析

|步骤|时间复杂度|说明|
|---|---|---|
|扁平化与排序|`O(n*m log(n*m))`|总共有 n*m 个元素，排序的时间复杂度为 O(N log N)|
|滑动窗口遍历|`O(n*m)`|每个元素最多进入窗口一次，离开窗口一次|
|总时间复杂度|`O(nm log(nm))`|对于 n*m=3e5，总操作数约为 3e5 * 19 ≈ 5.7e6，完全可以通过|

**空间复杂度：** `O(n*m)`，用于存储所有成员的信息和频次数组

## 完整代码
### ***C++:***

```CPP
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits> // 用于INT_MAX
using namespace std;

// 结构体：存储成员的战力值和所属战队编号
struct Member {
    int val;    // 战力值
    int team;   // 战队编号（0~n-1）
    
    // 重载小于运算符，用于按战力值从小到大排序
    bool operator<(const Member& other) const {
        return val < other.val;
    }
};

int main() {
    // 优化输入输出流：对于大数据量，不加这两句会超时
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int n, m;
    cin >> n >> m;
    
    // 存储所有成员的信息
    vector<Member> all_members(n * m);
    
    // 读取输入
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m; ++j) {
            int index = i * m + j;
            cin >> all_members[index].val;
            all_members[index].team = i;
        }
    }
    
    // 按战力值从小到大排序所有成员
    sort(all_members.begin(), all_members.end());
    
    vector<int> team_count(n, 0); // 记录当前窗口内每个战队的成员数量
    int left = 0;                 // 滑动窗口左指针
    int covered_teams = 0;        // 当前窗口内包含的不同战队数量
    int min_range = INT_MAX;      // 全局最小极差
    
    // 滑动窗口右指针遍历所有成员
    for (int right = 0; right < all_members.size(); ++right) {
        int current_team = all_members[right].team;
        
        // 如果该战队之前没有成员在窗口中，覆盖战队数加1
        if (team_count[current_team] == 0) {
            covered_teams++;
        }
        team_count[current_team]++;
        
        // 当窗口包含所有n个战队时，尝试收缩左边界以找到更小的极差
        while (covered_teams == n) {
            // 更新全局最小极差
            int current_range = all_members[right].val - all_members[left].val;
            if (current_range < min_range) {
                min_range = current_range;
            }
            
            // 移出左指针指向的成员
            int left_team = all_members[left].team;
            team_count[left_team]--;
            
            // 如果该战队在窗口内没有成员了，覆盖战队数减1
            if (team_count[left_team] == 0) {
                covered_teams--;
            }
            
            left++; // 左指针右移
        }
    }
    
    cout << min_range << endl;
    
    return 0;
}
```
