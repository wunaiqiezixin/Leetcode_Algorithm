# 18. 四数之和
[力扣题目链接](https://leetcode.cn/problems/4sum/description/)

---
### 快速排序--要特别熟悉
```C
//swap()函数,用于交换两个变量的值
void swap(int* a, int* b)
{
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

//快速排序
void quickSort(int* a, int start, int end)
{
    if (start >= end) return; //递归边界
    int k = a[start];
    int left = start, right = end;
    while (left < right)
    {
        while (left < right && a[right] >= k)
            right--;
        swap(&a[left], &a[right]);
        while (left < right && a[left] <= k)
            left++;
        swap(&a[left], &a[right]);
    }
    a[left] = k;
    quickSort(a, start, left - 1);
    quickSort(a, left + 1, end);
}
```
- 本题思路和[15.三数之和](https://leetcode.cn/problems/3sum/description/)类似
- 先用两重循环确定两个数，在通过双指针确定后两个数
- 难点在于剪枝和去重
- 注意：当四数相加时，其值可能溢出int范围，用long long类型较为稳妥

/**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume caller calls free().
 */
//swap()函数,用于交换两个变量的值
void swap(int* a, int* b)
{
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

//快速排序
void quickSort(int* a, int start, int end)
{
    if (start >= end) return; //递归边界
    int k = a[start];
    int left = start, right = end;
    while (left < right)
    {
        while (left < right && a[right] >= k)
            right--;
        swap(&a[left], &a[right]);
        while (left < right && a[left] <= k)
            left++;
        swap(&a[left], &a[right]);
    }
    a[left] = k;
    quickSort(a, start, left - 1);
    quickSort(a, left + 1, end);
}

int** fourSum(int* nums, int numsSize, int target, int* returnSize, int** returnColumnSizes) {
    *returnSize = 0;
    const int MAX_POSSIBLE = numsSize * numsSize;
    *returnColumnSizes = (int*)malloc(sizeof(int) * MAX_POSSIBLE);
    int** ret = (int**)malloc(sizeof(int*) * MAX_POSSIBLE);
    //排序数组nums
    quickSort(nums, 0, numsSize - 1);
    //通过两重循环确定两个数
    int i, j;
    for (i = 0; i < numsSize - 3; i++)
    {
        //剪枝
        if (nums[i] >= 0 && nums[i] > target) break;
        //对第一个数去重
        if (i > 0 && nums[i] == nums[i - 1]) continue;
        //确定第二个数
        for (j = i + 1; j < numsSize - 2; j++)
        {
            //剪枝
            if (nums[i] + nums[j] >= 0 && nums[i] + nums[j] > target) break;
            //对第二个数去重
            if (j > i + 1 && nums[j] == nums[j - 1]) continue;
            //双指针确定最后两个数
            int left = j + 1, right = numsSize - 1;
            while (left < right)
            {
                long long sum = (long long)nums[i] + nums[j] + nums[left] + nums[right];
                if (sum < target) left++;
                else if (sum > target) right--;
                else
                {
                    //对最后两个数去重
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    //记录结果
                    int* ans = (int*)malloc(sizeof(int) * 4);
                    ans[0] = nums[i];
                    ans[1] = nums[j];
                    ans[2] = nums[left];
                    ans[3] = nums[right];
                    ret[*returnSize] = ans;
                    (*returnColumnSizes)[*returnSize] = 4;
                    (*returnSize)++;
                    //移动left,right，寻找其他解
                    left++;
                    right--;
                }
            }
        }
        
    }
    return ret;
}
