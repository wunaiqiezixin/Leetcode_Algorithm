# 19. 删除链表的倒数第 N 个结点
[力扣题目链接](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/description/)
## 快慢指针
* 注意题目说明 1<=n<=size
* 设定虚拟头结点，不要考虑删除头结点的情况
* 初始快指针fast、慢指针slow均指向虚拟头结点
* fast先走n步
* fast、slow一起走，直到fast->next == NULL
* 注意是fast->next == NULL时，停止移动，确保slow指向要删除结点的前一个
### ***C***
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* removeNthFromEnd(struct ListNode* head, int n) {
    //设立虚拟头结点
    struct ListNode dummy;
    dummy.next = head;
    //双指针
    struct ListNode* fast = &dummy;
    struct ListNode* slow = &dummy;
    //快指针先走n步：
    while (n--)
    {
        fast = fast->next;
    }
    //快慢指针一起走，直到fast->next == NULL
    while (fast->next) /* n>=1 */
    {
        fast = fast->next;
        slow = slow->next;
    }
    //删除slow后面的结点
    struct ListNode* cur = slow->next;
    slow->next = slow->next->next;
    free(cur);
    //返回头结点
    return dummy.next;
}
```
