# 26.删除有序数组中的重复项
[力扣题目链接](https://leetcode.cn/problems/remove-duplicates-from-sorted-array/)
## 如何删除数组元素
<ol>
  <li>
    要删除下标为i的元素，将下标i+1往后的元素全部往前移动一位。(暴力法)
  </li>
  <li>
    运用双指针slow、fast。slow记录要保留元素的位置，fast从头寻找要保留的元素，每找到一个，则nums[slow++] = nums[fast++]。(双指针法)
  </li>
</ol>

## 暴力法
* 双重循环：外重循环用标识符i,内重循用环标识符j
* i 从 0 到 numsSize-2 用来寻找要删除的元素
* j 从 i 到 numsSize-2 用来将要删除的元素的后面元素全部往前移动一位
* 注意每次移动完成后，将 i-- 再进入下一重外重循环
* Python代码实现用while循环，不用for循环，方便每次移动后进行 i-=1
* C语言代码实现可以用for循环
