# 28. 找出字符串中第一个匹配项的下标
[力扣题目链接](https://leetcode.cn/problems/find-the-index-of-the-first-occurrence-in-a-string/description/)

---
>KMP算法

快速判断一个模式串是否在主串中    

从模式串和子串的起点开始匹配`p1`指向主串，`p2`指向模式串，如果字符匹配，则两个指针都右移一位     
当遇到不匹配时，保持`p1`不动，`p2`根据`next`数组回退 `p2 = next[p2]` (确保`p2 > 0`)   
重新比较`p1`和`p2`所指向的字符，如果相同，同时右移指针，不同则回退指针`p2`   
当指针`p2 <= 0` 无法回退时，保持`p2`不动， 将 `p1`指针右移一位   
循环迭代，如果最后 `p1` 指向模式串的尾部字符，则匹配成功，匹配的起点为 `p2 - p1`    

>计算next数组   

`next[i]` : 子串  [0, i - 1] 的前后缀最大相等长度    
`next[0] = next[1] = 0`    
定义指针 `int cur = 2, comp = 0`    
如果 `next[cur - 1] == next[comp]` ---> `next[cur] = comp + 1` (注意：不能将`comp`换成 `next[cur - 1]`, 因为`comp`可能会回退)      
如果不相等且`comp > 0` ---> 寻找更小的匹配， `comp`根据 `next`进行回退， 即 `comp = next[comp]`     
如果 `comp == 0`， 即`comp`无法回退 ， 则说明该子串的最大前后缀相等长度为0 (`next[cur++] = 0`)    



### ***CPP***
```CPP
class Solution {
private:
//计算next数组
vector<int> getNext(const string& needle)
{
    //如果长度小于2，返回0
    if (needle.size() < 2)
    {
        return {0};
    }
    vector<int> next(needle.size());
    next[0] = next[1] = 0;//前两项始终为0
    int cur = 2;
    int comp = next[cur - 1];
    while (cur < needle.size())
    {
        if (needle[cur - 1] == needle[comp]){//相等
            next[cur] = comp + 1;
            cur++;
            comp++;
        }else if (comp > 0){//不相等，comp可以回退
            comp = next[comp];
        }else{//不相等，comp无法回退
            next[cur++] = 0;
        }
    }
    return next;
}
public:
    int strStr(string haystack, string needle) {
        vector<int> next = getNext(needle);
        int p1 = 0, p2 = 0;//p1指向haystack, p2指向needle
        while (p1 < haystack.size() && p2 < needle.size())
        {
            if (haystack[p1] == needle[p2]){//相等
                p1++;
                p2++;
            }else if (p2 > 0){//不相等，p2回退
                p2 = next[p2];
            }else{//不相等，p2无法回退
                p1++;
            }
        }
        return p2 == needle.size() ? p1 - p2 : -1;
    }
};
```
