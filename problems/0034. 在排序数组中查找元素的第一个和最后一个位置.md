# 34. 在排序数组中查找元素的第一个和最后一个位置

[力扣题目链接](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/description/)

给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。

 

示例 1：

输入：nums = [5,7,7,8,8,10], target = 8

输出：[3,4]

示例 2：

输入：nums = [5,7,7,8,8,10], target = 6

输出：[-1,-1]

示例 3：

输入：nums = [], target = 0

输出：[-1,-1]
 

### 提示：

* 0 <= nums.length <= 105
* -109 <= nums[i] <= 109
* nums 是一个非递减数组
* -109 <= target <= 109

## 思路：

* 二分查找两次

* 第一次查找左边界，第二次查找有边界

* 以查找左边界为例：每当 nums[mid] = target 时，记录左边界的下标为 best = mid ,但我们不能保证这个值是最左边的，因此，再令 left = mid + 1 ,查找有无更大的best的值

### ***C：***
```C
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
int getLeft(int* nums,int numsSize,int target)
{
    int left = 0,right = numsSize - 1,best = -1;
    int mid;
    while (left <= right)
    {
        mid = left + (right - left) / 2;
        if (nums[mid] < target)
        {
            left = mid + 1;
        }
        else if (nums[mid] > target)
        {
            right = mid - 1;
        }
        else
        {
            best = mid;
            right = mid - 1;
        }
    }
    return best;
}
int getRight(int* nums,int numsSize,int target)
{
    int left = 0,right = numsSize - 1,best = -1;
    int mid;
    while (left <= right)
    {
        mid = left + (right - left) / 2;
        if (nums[mid] < target)
        {
            left = mid + 1;
        }
        else if (nums[mid] > target)
        {
            right = mid - 1;
        }
        else
        {
            best = mid;
            left = mid + 1;
        }
    }
    return best;
}
int* searchRange(int* nums, int numsSize, int target, int* returnSize) {
    *returnSize = 2;
    int leftBoundary = getLeft(nums,numsSize,target);
    int rightBoundary = getRight(nums,numsSize,target);
    int* result = (int*)malloc(sizeof(int) * 2);
    result[0] = leftBoundary;
    result[1] = rightBoundary;
    return result;
}
```
