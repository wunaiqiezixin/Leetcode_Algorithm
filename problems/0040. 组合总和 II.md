# 40. 组合总和 II
[力扣题目链接](https://leetcode.cn/problems/combination-sum-ii/description/)

---
## 🧠思路
- candidates 中的每个数字在每个组合中只能使用 一次 。
- 注意：解集不能包含重复的组合。
    
本题给出了一个可以有重复元素的数组，并且每个元素只能用一次     
因此，用一个```used```数组表示```candidates```中的元素是否已经使用   
```startIndex```用于表示```candidates```数组的起点    
本题的难点在于如何去重：
```C
if (i > 0 && candidates[i] == candidates[i - 1] && used[i - 1] == false) continue;
```
首先要对数组从小到大排序，如果当前元素和前一个元素相等并且前一个元素已经使用过，则该元素不能再使用，跳过   
这里的```used[i-1]==false```是上一层回溯过来的，所以```used[i - 1]```仍然为false 
### ***C:***
```C
/**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume caller calls free().
 */
//swap()函数,用于交换两个变量的值
void swap(int* a, int* b)
{
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

//快速排序
void quickSort(int* a, int start, int end)
{
    if (start >= end) return; //递归边界
    int k = a[start];
    int left = start, right = end;
    while (left < right)
    {
        while (left < right && a[right] >= k)
            right--;
        swap(&a[left], &a[right]);
        while (left < right && a[left] <= k)
            left++;
        swap(&a[left], &a[right]);
    }
    a[left] = k;
    quickSort(a, start, left - 1);
    quickSort(a, left + 1, end);
}
//used数组用来表示candidate中的元素是否用过
void backtracking(int* candidates, int candidatesSize, int target, int startIndex, bool* used, int** result, int* resultTop, int* path, int pathTop, int* colSizes)
{
    //终止条件
    if (target < 0) return;//和超过target
    if (target == 0)
    {
        //和等于target，找到一组解
        int* tmp = (int*)malloc(sizeof(int) * pathTop);
        for (int i = 0; i < pathTop; i++) tmp[i] = path[i];
        colSizes[(*resultTop)] = pathTop;
        result[(*resultTop)++] = tmp;
        return;
    }
    //递归回溯
    for (int i = startIndex; i < candidatesSize; i++)
    {
        //去重(重难点)
        if (i > 0 && candidates[i] == candidates[i - 1] && used[i - 1] == false) continue;

        //前序位置：
        path[pathTop++] = candidates[i];
        used[i] = true;
        backtracking(candidates, candidatesSize, target - candidates[i], i + 1, used, result, resultTop, path, pathTop, colSizes);
        //后序位置
        pathTop--;
        used[i] = false;
    }
}
int** combinationSum2(int* candidates, int candidatesSize, int target, int* returnSize, int** returnColumnSizes) {
    int** result = (int**)malloc(sizeof(int*) * 1000);
    *returnSize = 0;
    bool* used = (bool*)malloc(sizeof(bool) * 1000);
    for (int i = 0; i < 1000; i++) used[i] = false;
    int* path = (int*)malloc(sizeof(int) * 1000);
    *returnColumnSizes = (int*)malloc(sizeof(int) * 1000);
    if (!candidatesSize) return result;
    quickSort(candidates, 0, candidatesSize - 1);//排序数组
    backtracking(candidates, candidatesSize, target, 0, used, result, returnSize, path, 0, *returnColumnSizes);
    return result;
}
```
