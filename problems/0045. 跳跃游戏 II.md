# 45. 跳跃游戏 II
[力扣题目链接](https://leetcode.cn/problems/jump-game-ii/description/)

---
## 🧠思路
不用关心具体是怎么跳的，只要关注覆盖范围是否到达数组末尾     
变量`int step = 0`表示跳的步数      
变量`int curCover = 0`表示当前的覆盖范围     
变量`int nexCover = 0`表示下一次的(最大)覆盖范围(`nexCover = max(nexCover, nums[i] + i`)     
用一个指向数组的指针`i`遍历数组(for循环)   
当`i == curCover`，即到达当前覆盖范围的末尾时，       
首先判断`i == nums.size() - 1`(是否是数组的尾部， 如果是， break)     
如果`i != nums.size() - 1` , 更新覆盖范围(`curCover = nexCover`), 步数加一(`step++`)    
### ***C++***
```CPP
class Solution {
public:
    int jump(vector<int>& nums) {
        int step = 0;
        int curCover = 0;//当前的覆盖范围
        int nexCover = 0;//下一次的(最大)覆盖范围
        for (int i = 0; i < nums.size(); i++)
        {
            //记录下一次的最大覆盖范围
            nexCover = max(nexCover, nums[i] + i);
            
            if (i == curCover)//到达当前覆盖范围的尾部
            {
                if (i == nums.size() - 1)//为数组尾部
                    break;
                else
                {
                    step++;
                    curCover = nexCover;
                }
            }
        }
        return step;
    }
};
```
