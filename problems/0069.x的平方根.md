# 69.x的平方根
[力扣题目链接](https://leetcode.cn/problems/sqrtx/description/)

## 牛顿迭代法

### ***C***
```C
#define EPS 1e-7
int mySqrt(int x) {
    if (x == 0) return 0;
    double a = x / 2.0;
    double last;
    do {
        last = a;
        a = (a + x / a) / 2;
    } while (fabs(last - a) > EPS);
    return (int)a;
}
```
