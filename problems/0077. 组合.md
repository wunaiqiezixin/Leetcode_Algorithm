# 77. 组合
[力扣题目链接](https://leetcode.cn/problems/combinations/description/)

## 🧠思路
使用回溯法    
将其抽象为一棵多叉树    
而后我们以遍历多叉树的思维进行递归回溯    
本题因该抽象为一棵深度为```k```的二叉树     
我们会定义一个没有返回值的函数```backtracking()```    
函数的参数就代表了唯一处于哪个节点，而所谓的回溯，就是改变函数的参数，让其移动到某个节点(状态转移)    
可以将```backtracking()```看作一个指针，它递归遍历整颗树，当它遍历到叶子节点时收集答案    
再写程序时，我们将注意力放到一个节点上，只需关注它的**前序位置**和**后序位置**,因为**多叉树没有中序位置**    
在前序位置：做出决策    
在后序位置：撤销决策

```C
/**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume caller calls free().
 */
void backtracking(int n, int k, int startIndex, int* path, int pathTop, int** result, int* resultTop)
{
    //终止条件：叶子节点
    if (pathTop == k)
    {
        //拷贝path中的元素
        int* tmp = (int*)malloc(sizeof(int) * pathTop);
        for (int i = 0; i < pathTop; i++)
            tmp[i] = path[i];
        //收集结果
        result[(*resultTop)++] = tmp;
        return;
    }

    for (int i = startIndex; i <= n; i++)
    {
        path[pathTop++] = i;//前序位置
        backtracking(n, k, i + 1, path, pathTop, result, resultTop);
        pathTop--;//后序位置
    }
    return;
}

int** combine(int n, int k, int* returnSize, int** returnColumnSizes) {
    int** result = (int**)malloc(sizeof(int*) * 1000000);
    int* path = (int*)malloc(sizeof(int) * 1000000);
    *returnSize = 0;
    backtracking(n, k, 1, path, 0, result, returnSize);
    *returnColumnSizes = (int*)malloc(sizeof(int) * (*returnSize));
    for (int i = 0; i < (*returnSize); i++)
        (*returnColumnSizes)[i] = k;
    return result;
}
```
