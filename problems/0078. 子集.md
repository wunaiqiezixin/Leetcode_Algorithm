# 78. 子集
[力扣题目链接](https://leetcode.cn/problems/subsets/description/)

---
>需要在每个节点都收集答案     
>定义`startIndex`变量    
>当`startIndex >= nums.size()`时终止，注意在终止条件前收集答案

### ***C++***
```CPP
class Solution {
private:
    vector<vector<int>> ans;
    vector<int> path;
    void backtracking(vector<int>& nums, int startIndex)
    {
        //在前序位置收集结果(每个节点都要收集)
        ans.push_back(path);
        //终止条件
        if (startIndex >= nums.size()) return;
        //递归回溯
        for (int i = startIndex; i < nums.size(); i++)
        {
            path.push_back(nums[i]);
            backtracking(nums, i + 1);
            path.pop_back();
        }
    }
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        backtracking(nums, 0);
        return ans;
    }
};
```
