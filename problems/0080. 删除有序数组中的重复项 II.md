# 80. 删除有序数组中的重复项 II
[力扣题目链接](https://leetcode.cn/problems/remove-duplicates-from-sorted-array-ii/description/)

---
## 思路
>排序 + 双指针

### ***C++***
```CPP
/*
两个指针：
    1.slow：表示下标[0, slow)的元素要保留
    2.fast：遍历数组
*/
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        //数组长度小于等于2
        int n = nums.size();
        if (n <= 2)
            return n;
        //保证数组有序:sort(nums.begin(), nums.end());
        int slow, fast;
        slow = 2;//至少有两个元素可以保留
        //fast遍历一遍数组
        for (fast = 2; fast < n; ++fast)
        {
            //判断是否可以保留
            //即和[0, slow - 1]的元素比较，看是否重复
            if (nums[fast] != nums[slow - 2])//是否重复两次以上
                nums[slow++] = nums[fast];
        }
        return slow;
    }
};
```
