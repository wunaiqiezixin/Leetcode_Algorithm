# 82. 删除排序链表中的重复元素 II
[力扣题目链接](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/description/?envType=problem-list-v2&envId=linked-list)
* 头结点可能被删除，所以要有一个指向头结点的虚拟头结点
* cur指针指向虚拟头结点
* 要删除的是重复的至少两个或以上元素，所以要保证 cur->next && cur->next->next 存在
### ***C***
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* deleteDuplicates(struct ListNode* head) {
    //虚拟头结点
    struct ListNode dummy;
    dummy.next = head;
    struct ListNode* cur = &dummy;
    //遍历链表
    while (cur->next && cur->next->next)
    {
        if (cur->next->val == cur->next->next->val)
        {
            /*设立一个临时指针tmp指向cur->next,寻找cur结点后有几个结点的值相等*/
            struct ListNode* tmp = cur->next; //也方便将cur结点后的值相等的结点都free掉
            while (tmp->next && tmp->val == tmp->next->val)//注意 tmp != NULL
            {
                struct ListNode* p = tmp;
                tmp = tmp->next;
                free(p);
            }
            //最终tmp指向最后一个值等于cur->next->val的结点
            cur->next = tmp->next;
        }
        else
        {
            cur = cur->next;
        }
    }
    return dummy.next;
}
```
