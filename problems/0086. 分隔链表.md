# 86. 分隔链表
[力扣题目链接](https://leetcode.cn/problems/partition-list/description/?envType=problem-list-v2&envId=linked-list)
## 思路：
* 定义两个结构体Big,Small分别用来连接小于x,大于等于x的结点，将结点分成两部分
* 遍历一次链表，如果结点值小于x，连给Small,反之连给Big
* 最后将两部分合并，Small的尾部指向Big的头部
* 注意将Big的尾部设成NULL,避免成环之类的
### ***C***
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* partition(struct ListNode* head, int x) {
    //设立两个虚拟头结点
    struct ListNode Small; //链接小于x的结点
    struct ListNode Big;  //链接大于等于x的结点
    struct ListNode* small = &Small;
    struct ListNode* big = &Big;
    while (head)
    {
        if (head->val < x)
        {
            small->next = head;
            small = small->next;
        }
        else
        {
            big->next = head;
            big = big->next;
        }
        head = head->next;
    }
    small->next = Big.next; // 将两个大小链表重新组合
    big->next = NULL; //注意将big指针指向NULL，避免成环
    return Small.next;
}
```
