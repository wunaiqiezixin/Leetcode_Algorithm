# 1002. 查找共用字符
[力扣题目链接](https://leetcode.cn/problems/find-common-characters/description/)

---
- 1 <= words[i].length <= 100
- words[i] 由小写英文字母组成
### 用数组实现哈希表
- 用长度为26的整形数组表示26个小写字母的出现频率
- 用hashTable数组记录第一个单词的字母频率
```C
int hashTable[26] = {0};
int i;
for (i = 0;i < strlen(words[0]);i++)
{
    hashTable[words[0][i] - 'a']++;
}
```
- 用updataHashTable函数更新最小值
```C
void updataHashTable(int* hashTableOne, int* hashTableTwo)
{
    int i;
    for (i = 0;i < 26;i++}
    {
        if (hashTableOne[i] > hashTableTwo[i]) hashTableOne[i] = hashTableTwo[i];
    }
}
```
- 用newHashTable数组记录第一个单词以后的字母频率，并与hashTable数组进行比较，更新最小值
```C
int i;
for (i = 1;i < strlen(words[i];i++)
{
    int newHashTable[26] = {0};
    int j;
    for (j = 0;j < strlen(words[i]);j++)
    {
        newHashTable[words[i][j] - 'a']++;
    }
    updataHashTable(hashTable,newHashTable);
}
```

- 最后将hashTable转换为字符存入结果数组
```C
char** ans = (char**)malloc(sizeof(char*) * 100);
*returnSize = 0;
int i;
for (i = 0;i < 26;i++)
{

    {

    }
}
```

```C
void updataHashTable(int* hashTableOne, int* hashTableTwo) //将最小值更新，存放在hashTableOne中
{
    int i;
    for (i = 0;i < 26;i++)
    {
        if (hashTableOne[i] > hashTableTwo[i]) hashTableOne[i] = hashTableTwo[i];
    }
}

char** commonChars(char** words, int wordsSize, int* returnSize) {
    *returnSize = 0;
    char** ans = (char**)malloc(sizeof(char*) * 100);
    int hashTable[26] = {0}; //存放第一个单词的字母频率
    int i;
    for (i = 0;i < strlen(words[0]);i++)
    {
        hashTable[words[0][i] - 'a']++;
    }
    //将其与后面单词的字母频率比较，并更新最小值
    for (i = 1;i < wordsSize;i++)
    {
        int newHashTable[26] = {0};
        int j;
        for (j = 0;j < strlen(words[i]);j++)
        {
            newHashTable[words[i][j] - 'a']++;
        }
        //更新最小值
        updataHashTable(hashTable,newHashTable);
    }
    //将结果放入ans
    for (i = 0;i < 26;i++)
    {
        if (hashTable[i])
        {
            int j;
            for (j = 0;j < hashTable[i];j++)
            {
                char* tmp = (char*)malloc(sizeof(char) * 2);
                tmp[0] = i + 'a';
                tmp[1] = '\0';
                ans[(*returnSize)++] = tmp;
            }
        }
    }
    return ans;
}
```
