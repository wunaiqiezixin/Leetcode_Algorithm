# 121. 买卖股票的最佳时机
[力扣题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/description/)  

---
>动态规划

### ***C++***
```CPP
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int days = prices.size();
        /*
            dp[i][0]: 第i天持有股票的最大利润
            dp[i][1]：第i天不持有股票的最大利润
        */
        vector<vector<int> > dp(days, vector<int>(2, 0));
        /* 初始化 */
        dp[0][0] = 0 - prices[0];/* 第0天持有股票，即第0天买入股票 */
        dp[0][1] = 0; /* 第0天不持有股票，即第0天没有买入股票 */
        int i;
        for (i = 1; i < days; ++i)
        {
            dp[i][0] = max(
                dp[i - 1][0], /* 第(i - 1)天已经持有股票 */
                0 - prices[i] /* 第i天买入股票 */
            );
            dp[i][1] = max(
                dp[i - 1][1], /* 第(i - 1)天就已经不持有股票了 */
                dp[i - 1][0] + prices[i] /* 第i天卖出股票 */
            );
        }

        return dp[days - 1][1];
    }
};
```
