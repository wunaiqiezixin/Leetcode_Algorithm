# 134. 加油站
[力扣题目链接](https://leetcode.cn/problems/gas-station/description/)

---
>贪心算法
## 思路
`gas[i] - cost[i]`为第i个加油站的油量增量(可以为负数)     
`start`表示出发的加油站下标     
`curSum`表示从出发点到当前加油站的油量增量     
`totSum`表示遍历完一遍数组后总的油量增量，如果为负数则表示无论从哪一个加油站出发均不能回到起点，返回-1       
`curSum < 0` 表明从0到i的加油站均不能作为起点， 更新`start = i + 1`

### ***C++***
```CPP
class Solution {
public:
    int canCompleteCircuit(vector<int>& gas, vector<int>& cost) {
        int totSum = 0;
        int curSum = 0;//从出发点到当前加油站的剩余油量
        int start = 0;//出发点
        for (int i = 0; i < gas.size(); i++)
        {
            curSum += gas[i] - cost[i];
            totSum += gas[i] - cost[i];
            if (curSum < 0)
            {
                start = i + 1;
                curSum = 0;
            }
        }
        if (totSum < 0) return -1;
        return start;
    }
};
```
