# 135. 分发糖果
[力扣题目链接](https://leetcode.cn/problems/candy/description/)

---
## 思路
先处理左边孩子比右边孩子得分高的情况(从左往右遍历数组)     
再处理右边孩子比左边孩子得分高的情况(从右往左遍历数组)    

### ***C++***
```CPP
class Solution {
public:
    int candy(vector<int>& ratings) {
        vector<int> candy(ratings.size(), 1);
        //从左往右遍历，处理右边孩子比左边孩子得分高的情况
        for (int i = 1; i < ratings.size(); i++)
        {
            if (ratings[i] > ratings[i - 1])
            {
                candy[i] = candy[i - 1] + 1;
            }
        }
        //再从右往左遍历，处理左孩子比右孩子得分高的情况
        for (int i = ratings.size() - 2; i >= 0; i--)
        {
            if (ratings[i] > ratings[i + 1])
            {
                candy[i] = max(candy[i], candy[i + 1] + 1);//两种情况都要满足，取最大值
            }
        }
        //统计糖果数量
        int ans = 0;
        for (int i = 0; i < ratings.size(); i++)
        {
            ans += candy[i];
        }
        return ans;
    }
};
```
