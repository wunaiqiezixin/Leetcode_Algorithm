# 151. 反转字符串中的单词
[力扣题目链接](https://leetcode.cn/problems/reverse-words-in-a-string/description/)
---
---
### 思路：
- 先去除多余的空格：
>用快慢指针
>
>因为两个单词中间有且仅有一个空格，所以但快指针指向空个并且快指针的下一个也为空格时，直接跳过，快指针进入下一重循环
```C
//去除多余的空格
void removeExtraSpace(char* s)
{
    int left = 0, right = strlen(s) - 1;
    //除去前置空格
    while (s[left] == ' ') left++;
    //除去后置空格
    while (s[right] == ' ') right--;
    //去除中间多余的空格
    int fast = left;
    int slow = 0;
    for (;fast <= right;fast++)
    {
        //如果当前为空格且下一个也为空格，跳过
        if (s[fast] == ' ' && s[fast+1] == ' ')
            continue;
        s[slow++] = s[fast];//否则，加到下标slow处
    }
    s[slow] = '\0';//在末尾添加空字符
}
```
- 然后整体反转题目给的字符串,自己定义一个reverse函数
```C
//在给定范围内反转字符串
void reverse(char* s, int left, int right)
{
    while (left < right)
    {
        char tmp = s[left];
        s[left++] = s[right];
        s[right--] = tmp;
    }
}
```
- 最后反转字符串中的每一个单词，主要是找到每个单词的起点和终点
```C
char* reverseWords(char* s) {
    removeExtraSpace(s);
    reverse(s,0, strlen(s) - 1);
    //反转字符串中的每一个单词
    int start = 0;
    int end = 0;
    for (;end <= strlen(s);end++)
    {
        if (s[end] == ' ' || s[end] == '\0')
        {
            reverse(s, start, end - 1);
            start = end + 1;//调整单词起始，让其指向下一个单词的开头
        }
    }
    return s;
}
```
