# 188. 买卖股票的最佳时机 IV
[力扣题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/description/)

---
## 思路
>三维dp
1. dp数组的含义：  
`dp[i][0][j]` 第i天第j次买入股票的最大利润  
`dp[i][1][j]` 第i天第j次卖出股票的最大利润  
2. dp数组的初始化  
`dp[0][0][j] = -prices[0]` ：第0天，不论是第几次买入股票，成本都是`prices[0]`(在同一天重复买入卖出)  
`dp[0][1][j] = 0` ：第0天，不论是第几次卖出股票，利润都是0，因为重复的买入卖出，买卖了个寂寞  
3. 动态转移方程  
`dp[i][0][j] = max( dp[i - 1][0][j], dp[i - 1][1][j - 1] - prices[i] )`  
`dp[i][1][j] = max( dp[i - 1][1][j], dp[i - 1][0][j] + prices[i] )`  
4. 遍历顺序  
i - 1 -> i  
j - 1 -> j  
遍历顺序任意  

### ***C++***
```CPP
class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int len = prices.size();
        /*
            dp[i][0][j]:第i天第j次买入股票
            dp[i][1][j]:第i天第j次卖出股票
        */
        vector<vector<vector<int> > > dp(len, vector<vector<int> >(2, vector<int>(k + 1, 0)));
        int i, j;
        for (j = 0; j <= k; ++j)
            dp[0][0][j] = -prices[0];
        for (i = 1; i < len; ++i)
        {
            for (j = 1; j <= k; ++j)
            {
                dp[i][0][j] = max(
                    dp[i - 1][0][j],
                    dp[i - 1][1][j - 1] - prices[i]
                );
                dp[i][1][j] = max(
                    dp[i - 1][1][j],
                    dp[i - 1][0][j] + prices[i]
                );
            }
        }
        int maxProfit = 0;
        for (j = 0; j <= k; ++j)
        {
            if (dp[len - 1][1][j] > maxProfit)
                maxProfit = dp[len - 1][1][j];
        }
        return maxProfit;
    }
};
```

>优化空间
```CPP
class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int len = prices.size();
        /*
            dp[0][j]:第i天第j次买入股票
            dp[1][j]:第i天第j次卖出股票
        */
        vector<vector<int> > dp(2, vector<int>(k + 1, 0));
        int i, j;
        for (j = 0; j <= k; ++j)
            dp[0][j] = -prices[0];
        for (i = 1; i < len; ++i)
        {
            for (j = 1; j <= k; ++j)
            {
                dp[0][j] = max(
                    dp[0][j],
                    dp[1][j - 1] - prices[i]
                );
                dp[1][j] = max(
                    dp[1][j],
                    dp[0][j] + prices[i]
                );
            }
        }
        int maxProfit = 0;
        for (j = 0; j <= k; ++j)
        {
            if (dp[1][j] > maxProfit)
                maxProfit = dp[1][j];
        }
        return maxProfit;
    }
};
```
