# 198. 打家劫舍
[力扣题目链接](https://leetcode.cn/problems/house-robber/description/)

---

> dp[i]: 在[0, i]的房间偷窃， 能偷窃到的最大金额
> dp[i] = max(偷第i间房间的最大金额， 不偷第i间房间的最大金额)

```CPP
class Solution {
public:
    int rob(vector<int>& nums) {
        if (nums.size() == 1)
            return nums[0];
        //dp[i]: 在[0, i]的房屋偷，能偷盗的最大金额
        vector<int> dp(nums.size(), 0);
        //初始化dp数组
        dp[0] = nums[0];
        dp[1] = max(dp[0], nums[1]);
        //递推
        int i;
        for (i = 2; i < nums.size(); ++i)
            dp[i] = max(dp[i - 1], dp[i - 2] + nums[i]);
        return dp[nums.size() - 1];
    }
};
```
