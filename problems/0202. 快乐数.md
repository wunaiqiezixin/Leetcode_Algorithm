# 202. 快乐数
- 基本功，定义函数get_sum,得到各位的平方和
```C
int get_sum(int n)
{
    int sum = 0;
    do{
    sum += (n % 10) * (n % 10);
    n /= 10;
    }while (n);
    return sum;
```
- 若n不是快乐数，则在循环的过程中，各位的平方和会重复出现，形成了一个圈。我们可以用快慢指针法，快指针每次走两步，慢指针每次走一步，若快指针追上了慢指针，则说明，n不是快乐数
```C
int fast = n,slow = n;
do{
   slow = get_sum(slow);
   fast = get_sum(get_sum(fast));
   if (fast == 1 || slow == 1) return true;
} while (fast != slow);
return false;
```

## 完整代码
### ***C:***
```C
int get_sum(int n) //得到n各位的平方和
{
    int sum = 0;
    do{
        sum += (n % 10) * (n % 10);
        n /= 10;
    }while (n);
    return sum;
}

bool isHappy(int n) {
    //快慢指针法
    int slow = n;
    int fast = n;
    do{  //特别考虑：n = 1  n = 10
        slow = get_sum(slow);
        fast = get_sum(get_sum(fast));
        if (fast == 1 || slow == 1) return true;
    }while (fast != slow);
    return false;
}
```
