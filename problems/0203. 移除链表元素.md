# 203. 移除链表元素
[力扣题目链接](https://leetcode.cn/problems/remove-linked-list-elements/description/?envType=problem-list-v2&envId=linked-list)

## 思路：
* 要移除链表中的某个结点，就要定位到其前一个结点
* 而头结点无前驱结点，所以，移除链表的结点分两种情况
* ***移除头结点***、***移除非头结点***
### 方法一(分类讨论)：
* 移除头结点
* 移除非头结点
### ***C***
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* removeElements(struct ListNode* head, int val) {
    //移除头结点
    while (head != NULL && head->val == val)
    {
        struct ListNode* cur = head;
        head = head->next;
        free(cur);
    }
    //移除非头结点
    struct ListNode* ptr = head;
    while (ptr != NULL && ptr->next != NULL)
    {
        if (ptr->next->val == val)
        {
            struct ListNode* cur = ptr->next;
            ptr->next = ptr->next->next;
            free(cur);
        }
        else
        {
            ptr = ptr->next;
        }
    }
    return head;
}
```
### 方法二(构造一个虚拟头结点)
* 不用再进行分类讨论
### ***C***
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* removeElements(struct ListNode* head, int val) {
    struct ListNode dummy;
    dummy.next = head;
    struct ListNode* ptr = &dummy;
    while (ptr->next != NULL)
    {
        if (ptr->next->val == val)
        {
            struct ListNode* cur = ptr->next;
            ptr->next = ptr->next->next;
            free(cur);
        }
        else
        {
            ptr = ptr->next;
        }
    }
    return dummy.next;
}
```
## 递归
* 递归的边界为 head == NULL
* 若 head->val == val 则removeElements()应传入head->next为参数
* 若 head->val != val 则removeElements()应传入head为参数
### ***C***
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* removeElements(struct ListNode* head, int val) {
    if (head == NULL) return head;

    if (head->val == val)
    {
        struct ListNode* newHead = removeElements(head->next,val);
        free(head);
        return newHead;
    }
    else
    {
        head->next = removeElements(head->next,val);
        return head;
    }
}
```
### 另一种写法
```C
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
struct ListNode* removeElements(struct ListNode* head, int val) {
    if (head == NULL)
    {
        return NULL;
    }
    head->next = removeElements(head->next,val);
    return head->val == val ? head->next : head;
}
```
