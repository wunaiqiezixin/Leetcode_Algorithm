# 209. 长度最小的子数组

[力扣题目链接](https://leetcode.cn/problems/minimum-size-subarray-sum/)

给定一个含有 n 个**正整数的数组**和一个正整数 target 。

找出该数组中满足其总和大于等于 target 的长度最小的 子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

示例 1：

输入：target = 7, nums = [2,3,1,2,4,3]

输出：2

解释：子数组 [4,3] 是该条件下的长度最小的子数组。

示例 2：

输入：target = 4, nums = [1,4,4]

输出：1

示例 3：

输入：target = 11, nums = [1,1,1,1,1,1,1,1]

输出：0
 

提示：

* 1 <= target <= 109
* 1 <= nums.length <= 105
* 1 <= nums[i] <= 104
 

进阶：

如果你已经实现 O(n) 时间复杂度的解法, 请尝试设计一个 O(n log(n)) 时间复杂度的解法。

## 暴力枚举
* 用 i 表示子数组的起始位置，用 j 表示子数组的终止位置
* i 范围[0,numsSize-1], j 范围[i,nunsSize-1]
* length,minLength 记录当前符合条件的子数组的长度和最小长度
* 初始化minLength = numsSize + 1;

### ***C:***
```C
int minSubArrayLen(int target, int* nums, int numsSize) {
    int minLength = numsSize + 1;
    int i,j,length,Sum;

    for (i = 0;i < numsSize;i++)
    {
        Sum = 0;
        for (j = i;j < numsSize;j++)
        {
            Sum += nums[j];
            if (Sum >= target)
            {
                length = j - i + 1;
                if (minLength > length)
                {
                    minLength = length;
                }
            }
        }
    }
    return minLength == numsSize + 1 ? 0 : minLength;
}
```

## 滑动窗口-->剪枝思想
* left为窗口的左边界，right为窗口的右边界
* 1.若 [left,right] 满足状态（数组和大于等于target），则[left,right+1 ..+2..end] 也满足状态。可以将这类情况剪枝
* 遍历时右移left
* 2.若 [left,right] 不满足状态（数组和小于target），则[left+1...+2...end,right] 也不满足状态。可以将这类情况剪枝
* 遍历时右移right


### ***C:***
```C
int minSubArrayLen(int target, int* nums, int numsSize) {
    int i = 0,minLength = numsSize + 1,Sum = 0;
    int j,length;
    for (j = 0;j < numsSize;j++)
    {
        Sum += nums[j];
        while (Sum >= target)
        {
            length = j - i + 1;
            if (minLength > length)
            {
                minLength = length;
            }
            Sum -= nums[i++];
        }
    }
    return minLength != numsSize + 1 ? minLength : 0;
}
```
