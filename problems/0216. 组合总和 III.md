# 216. 组合总和 III
[力扣题目链接](https://leetcode.cn/problems/combination-sum-iii/description/)

## 🧠思路
重点就是画出多叉树，递归遍历多叉树，注意回溯逻辑   
来看多叉树的根节点，可选范围是```1~9```:说明有9个孩子    
如果取1,那么可选范围变为```2~9```(第一个孩子)   
如果取2,那么可选范围变为```3~9```(第二个孩子)    
.......................................................     
如果取i,那么可选范围变为```(i+1)~9```(第i个孩子)   

>也就是说，每个节点的可选范围终点都是9，但是起点要根据它是父亲的第几个孩子来决定，因此要维护一个```startIndex```变量表示起点      


从一个节点转移到另一个节点时，我们直接用```n-i```,不再维护变量```sum```，最后当遍历到树的第```k```层时，如果```n==0```，则说明这是一组解     
我们可以做一个减枝的操作，当```n < 0```时，直接返回
### ***C:***
```C
/**
 * Return an array of arrays of size *returnSize.
 * The sizes of the arrays are returned as *returnColumnSizes array.
 * Note: Both returned array and *columnSizes array must be malloced, assume caller calls free().
 */
void backtracking(int k, int n, int startIndex, int** result, int* resultTop, int*path, int pathTop)
{
    if (n < 0)//剪枝
        return;
    //终止条件：叶子节点
    if (pathTop == k)
    {
        //和是n
        if (n == 0)
        {
            //拷贝元素并收集结果
            int* tmp = (int*)malloc(sizeof(int) * pathTop);
            for (int i = 0; i < pathTop; i++)
                tmp[i] = path[i];
            result[(*resultTop)++] = tmp;
        }
        //若和不是n，直接retur
        return;
    }

    //递归遍历
    for (int i = startIndex; i <= 9; i++)
    {
        //前序位置：
        path[pathTop++] = i;
        backtracking(k, n - i, i + 1, result, resultTop, path, pathTop);
        //后序位置：
        pathTop--;
    }
    return;
}

int** combinationSum3(int k, int n, int* returnSize, int** returnColumnSizes) {
    int** result = (int**)malloc(sizeof(int*) * 10000000);
    *returnSize = 0;
    int* path = (int*)malloc(sizeof(int) * k);
    backtracking(k, n, 1, result, returnSize, path, 0);
    *returnColumnSizes = (int*)malloc(sizeof(int) * (*returnSize));
    for (int i = 0; i < *returnSize; i++)
        (*returnColumnSizes)[i] = k;
    return result;
}
```
