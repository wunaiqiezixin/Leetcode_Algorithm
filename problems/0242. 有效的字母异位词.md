# 242. 有效的字母异位词
[力扣题目链接](https://leetcode.cn/problems/valid-anagram/description/)

---
## 哈希法
### ***C***
```C
bool isAnagram(char* s, char* t) {
    int hash[26]; //索引0~25分别表示a~z,其值表示出现的次数

    for (int i = 0;s[i];i++)
    {
        hash[s[i] - 'a']++; //减去'a'，可以看作是哈希函数
    }
    for (int i = 0;t[i];i++)
    {
        hash[t[i] - 'a']--; //看能否抵消为0
    }
    for (int i = 0;i < 26;i++)
    {
        if (hash[i] != 0) return false;
    }
    return true;
}
```
