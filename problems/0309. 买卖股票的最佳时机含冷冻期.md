# 309. 买卖股票的最佳时机含冷冻期
[力扣题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-cooldown/description/)

---
## 思路
>动态规划  

1. dp数组的含义
dp[i][0]: 第i天，持有股票的状态  
dp[i][1]：第i天，保持卖出股票的状态(已经度过冷冻期)  
dp[i][2]：第i天，今天卖出股票  
dp[i][3]：第i天，今天为冷冻期  

2. dp数组的初始化   
`dp[0][0] = -prices[0];`
其余的为了递推公式，初始化为0  

3. 状态转移    
第i天持有股票 = max( 第i-1天就持有股票，第i-1天保持卖出股票的状态且第i天买入股票，第i-1天为冷冻期且第i天买入股票 )   
第i天保持卖出股票的状态 = max( 第i-1天保持卖出股票的状态，第i-1天为冷冻期 )    
第i天卖出股票 = 第i-1天持有股票的状态且第i天卖出股票   
第i天为冷冻期 = 第i-1天卖出股票   

4. 遍历顺序   
第i天的状态只能由第i-1天推出   
从左到右遍历   

### ***C++***
```CPP
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        /*
        dp[i][j]:第i天,状态j
            j = 0, 持有股票的状态
            j = 1, 保持卖出股票的状态
            j = 2, 今天卖出股票
            j = 3, 今天是冷冻期
        */
        vector<vector<int> > dp(prices.size(), vector<int>(4, 0));
        dp[0][0] = 0 - prices[0];
        
        for (int i = 1; i < prices.size(); ++i)
        {
            dp[i][0] = max(dp[i - 1][0], max(dp[i - 1][1] - prices[i], dp[i - 1][3] - prices[i]));
            dp[i][1] = max(dp[i - 1][1], dp[i - 1][3]);
            dp[i][2] = dp[i - 1][0] + prices[i];
            dp[i][3] = dp[i - 1][2];
        }
        return max(dp[prices.size() - 1][1], max(dp[prices.size() - 1][2], dp[prices.size() - 1][3]));
    }
};
```
