# 347. 前 K 个高频元素
[力扣题目链接](https://leetcode.cn/problems/top-k-frequent-elements/description/)

---
## 思路
用`unordered_map`来统计数字出现频率(数字 : 出现次数)  
用`priority_queue`来选出前k个高频数字  
从左往右遍历数组，将遍历到的每一个数组元素入堆，当堆中的元素大于k时，弹出堆顶元素(因此要维护的是一个小顶堆)  
最后遍历完一遍数组后，留下的就是数组中的前k个高频元素  

### ***C++***
```CPP
class Solution {
private:
    struct mycomprison {
        bool operator() (const pair<int, int> & a1, const pair<int, int> & a2) const {
            return a1.second > a2.second;//小顶堆
        }
    };
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        unordered_map<int, int> map; //统计频率
        for (int i = 0; i < nums.size(); i++)
        {
            map[nums[i]]++;
        }
        priority_queue<pair<int, int>, vector<pair<int, int>>, mycomprison> pri_que;//优先队列，选出Top-K
        for (unordered_map<int, int>::iterator i = map.begin(); i != map.end(); i++)
        {
            pri_que.push(*i);
            if (pri_que.size() > k)//只留下最大的k个数
                pri_que.pop();
        }
        vector<int> ans(k);
        //收集结果
        for (int i = 0; i < k; i++)
        {
            ans[i] = pri_que.top().first;
            pri_que.pop();
        }
        return ans;
    }
};
```
