# 367. 有效的完全平方数
[力扣题目链接](https://leetcode.cn/problems/valid-perfect-square/description/)

### ***C++***
>今天是 2026-05-06

```CPP
class Solution {
public:
    bool isPerfectSquare(int num) {
        int left = 1, right = min(num, (const int)sqrt(INT_MAX));
        while (left <= right)
        {
            int m = left + (right - left) / 2;
            if (m * m < num)
                left = m + 1;
            else if (m * m > num)
                right = m - 1;
            else 
                return true;
        }
        return false;
    }
};
```
