# 377. 组合总和 Ⅳ
[力扣题目链接](https://leetcode.cn/problems/combination-sum-iv/description/)

---
## 思路
>完全背包问题

本题求的是排列数，所以我们要先遍历背包，再遍历物品

### ***C++***
```CPP
class Solution {
public:
    int combinationSum4(vector<int>& nums, int target) {
        /* dp[j]: 恰好装满容量为j的背包的取法总数 */
        vector<unsigned int> dp(target + 1, 0);
        /* dp数组的初始化 */
        dp[0] = 1;
        /* 求排列数，先遍历背包再遍历物品 */
        for (int j = 1; j <= target; ++j)
            for (int i = 0; i < nums.size(); ++i)
                if (j >= nums[i])
                    dp[j] += dp[j - nums[i]];
        
        return dp[target];
    }
};
```
