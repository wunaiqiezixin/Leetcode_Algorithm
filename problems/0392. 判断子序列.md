# 392. 判断子序列
[力扣题目链接](https://leetcode.cn/problems/is-subsequence/description/)

---
>动态规划

### ***C++***
```CPP
class Solution {
public:
    bool isSubsequence(string s, string t) {
        //dp[i][j]:s以i-1结尾，t以j-1结尾的最长公共子序列
        vector<vector<int> > dp(s.size() + 1, vector<int>(t.size() + 1, 0));
        int i, j;
        for (i = 1; i <= s.size(); ++i)
        {
            for (j = 1; j <= t.size(); ++j)
            {
                if (s[i - 1] == t[j - 1])
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                else    
                    dp[i][j] = dp[i][j - 1];
            }
        }
        return dp[s.size()][t.size()] == s.size();
    }
};
```
