# 406. 根据身高重建队列
[力扣题目链接](https://leetcode.cn/problems/queue-reconstruction-by-height/description/)

---
## 思路
本题要将一个二维数组`people`的元素按一定的规则将其放入另一个二维数组`que`中  
`people`中每个元素都是有两个元素的数组，第一个`h`表示身高，第二个`k`表示前面有k个人的身高大于等于自己  
不要两个层面同时考虑，先定一个，随后再定另一个  
1. 将`people`根据身高从大到小排序，身高相同的k值小的在前
2. 从左到右遍历一遍`people`, 根据每一个元素的k值插入到`que`中

### ***C++***
```CPP
class Solution {
private:
    struct mycomprison {
        bool operator() (const vector<int> & a1, const vector<int> & a2) const {
            if (a1[0] == a2[0])
                return a1[1] < a2[1];//身高同，k小的在前
            return a1[0] > a2[0]; //高的在前
        }
    };
public:
    vector<vector<int>> reconstructQueue(vector<vector<int>>& people) {
        vector<vector<int>> que;
        sort(people.begin(), people.end(), mycomprison());//按身高从大到小排序
        //从左往右遍历数组，根据k调整数组顺序
        for (int i = 0; i < people.size(); i++)
        {
            int k = people[i][1];
            que.insert(que.begin() + k, people[i]);
        }
        return que;
    }
};
```
