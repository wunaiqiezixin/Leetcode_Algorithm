# 416. 分割等和子集
[力扣题目链接](https://leetcode.cn/problems/partition-equal-subset-sum/description/)

---
## 思路
01背包的变形  
`target = sum / 2`  
有`nums.size()`件物品，每件物品的价值和重量相等`nums[i]`  
能否从中取出价值为`target`的物品  

### ***C++***
```CPP
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int sum = accumulate(nums.begin(), nums.end(), 0);
        if (sum % 2) 
            return false;
        int target = sum / 2;

        vector<int> dp(target + 1, 0);/* dp[j]:容量为j的背包的最大价值*/

        for (int i = 0; i < nums.size(); i++)
        {
            for (int j = target; j >= nums[i]; j--)
                dp[j] = max(dp[j], dp[j - nums[i]] + nums[i]);
        }
        
        return dp[target] == target;
    }
};
```
