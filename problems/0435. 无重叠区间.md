# 435. 无重叠区间
[力扣题目链接](https://leetcode.cn/problems/non-overlapping-intervals/description/)  

---
## 思路
将数组`intervals`按左边界从小到大排序  
从左到右遍历`intervals`，当当前区间的左边界小于前一个区间的右边界时，说明这两个区间有重叠，需要删除的区间数量加1  
注意当前区间的右边界取这两个区间的右边界的最小值，因为还要判断当删除了当前区间后是否还会和下一个区间出现重叠  

### ***C++***
```CPP
class Solution {
private:
    struct mycomprison {//排序结构体：左边界从小到大
        bool operator() (const vector<int> & v1, const vector<int> & v2) const {
            if (v1[0] == v2[0])
                return v1[1] < v2[1];//左边界同，右边界小的在前
            return v1[0] < v2[0];
        }
    };
public:
    int eraseOverlapIntervals(vector<vector<int>>& intervals) {
        int ans = 0;//需要删除的区间数量
        sort(intervals.begin(), intervals.end(), mycomprison());
        for (int i = 1; i < intervals.size(); i++)
        {
            if (intervals[i - 1][1] > intervals[i][0])
            {
                ans++;
                intervals[i][1] = min(intervals[i - 1][1], intervals[i][1]);//更新第i个区间的右边界，判断是否还和下一个区间重叠
            }
        }
        return ans;
    }
};
```
