# 452. 用最少数量的箭引爆气球
[力扣题目链接](https://leetcode.cn/problems/minimum-number-of-arrows-to-burst-balloons/description/)

---
## 思路
对`points`数组按照左边界从小到大排序  
从左(第1个，不是第0个)往右遍历数组`points`   
如果第i个和第i-1个区间重叠，更新第i个区间的右边界(取第i个和第i-1个右边界的最小值)以便判断是否和下一个区间也有重叠部分  


### ***C++***
```CPP
class Solution {
private:
    struct mycomprison {
        bool operator() (const vector<int> & v1, const vector<int> & v2) const {
            if (v1[0] == v2[0])
                return v1[1] < v2[1];
            return v1[0] < v2[0];
        }
    };
public:
    int findMinArrowShots(vector<vector<int>>& points) {
        int ans = 1;//至少要用一个弓箭(points.size() >= 1)
        sort(points.begin(), points.end(), mycomprison());//按左边界从小到大排序
        for (int i = 1; i < points.size(); i++)
        {
            if (points[i][0] > points[i - 1][1]){
                ans++;
            }else{
                points[i][1] = min(points[i][1], points[i - 1][1]);//更新右边界，判断与下一个气球是否也重叠
            }
        }
        return ans;
    }
};
```
