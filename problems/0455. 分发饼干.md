# 455. 分发饼干
[力扣题目链接](https://leetcode.cn/problems/assign-cookies/description/)

---
>贪心算法
## 🧠思路
将饼干数组`s`和小孩胃口数组`g`进行排序   
指针`i`指向数组`g`,指针`j`指向数组`s`    
将小饼干优先分给胃口小的孩子    
### ***C++***
>while循环
```CPP
class Solution {
public:
    int findContentChildren(vector<int>& g, vector<int>& s) {
        int i = 0, j = 0;//指针
        int total = 0;//计数
        sort(g.begin(), g.end());
        sort(s.begin(), s.end());
        while (i < g.size() && j < s.size())
        {
            //小饼干优先给胃口小的孩子
            if (s[j] >= g[i])
            {
                i++;
                j++;
                total++;
            }
            else j++;
        }
        return total;
    }
};
```
>for循环
```CPP
class Solution {
public:
    int findContentChildren(vector<int>& g, vector<int>& s) {
        sort(s.begin(), s.end());
        sort(g.begin(), g.end());
        int i = 0, j = 0, total = 0;
        for (;j < s.size() && i < g.size(); j++)
        {
            if (s[j] >= g[i])
            {
                total++;
                i++;
            }
        }
        return total;
    }
};
```
