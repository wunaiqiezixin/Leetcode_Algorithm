# 503. 下一个更大元素 II
[力扣题目链接](https://leetcode.cn/problems/next-greater-element-ii/description/)

---
>单调栈+循环数组

## 思路
循环数组的处理：  
将数组加倍，从后往前遍历加倍后的数组，可以将加倍后的数组当成一个线性数组处理  
在写代码的时候也不是真的要将数组加倍了，我们从后往前遍历两遍数组就可以了  
令数组的长度为`n`, 那么我们只需要从`(2n-1) % n` 到 `0 % n` 遍历(加倍)数组  

### ***C++***
```CPP
class Solution {
public:
    vector<int> nextGreaterElements(vector<int>& nums) {
        int n = nums.size();
        vector<int> nextGreater(n, -1);
        stack<int> stk;
        int i;
        /* 数组加倍 */
        for (i = 2 * n - 1; i >= 0; --i)
        {
            /* 在栈中寻找下一个更大的元素 */
            while (!stk.empty() && nums[i % n] >= stk.top())
                stk.pop();
            if (!stk.empty())
                nextGreater[i % n] = stk.top();
            stk.push(nums[i % n]);
        }
        return nextGreater;
    }
};
```
