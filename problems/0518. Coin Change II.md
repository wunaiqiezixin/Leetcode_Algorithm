# 518. Coin Change II
[力扣题目链接](https://leetcode.cn/problems/coin-change-ii/description/)

---
### ***c++***

```CPP
class Solution {
public:
    int change(int amount, vector<int>& coins) {
        int i, j;
        vector<unsigned int> dp(amount + 1, 0);
        dp[0] = 1;
        for (i = 0; i < coins.size(); ++i)
            for (j = coins[i]; j <= amount; j++)
                dp[j] += dp[j - coins[i]];
        return dp[amount];
    }
};
```
