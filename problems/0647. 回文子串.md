# 647. 回文子串
[力扣题目链接](https://leetcode.cn/problems/palindromic-substrings/description/)

---
>动态规划

dp[i][j]表示下标i到下标j是否是回文子串  
注意遍历顺序，由下到上，由左到右  
因为递推公式中含:dp[i][j] = dp[i + 1][j - 1]  
将数组初始化为false  

### ***C++***
```CPP
class Solution {
public:
    int countSubstrings(string s) {
        int result = 0;
        //dp[i][j]:下标i到下标j的是不是s的回文子串
        vector<vector<bool> > dp(s.size(), vector<bool>(s.size(), false));//默认为false
        for (int i = s.size() - 1; i >= 0; --i)//由下至上
        {
            for (int j = i; j < s.size(); ++j)//由左至右
            {
                if (s[i] == s[j])
                {
                    if (j - i <= 1)
                        dp[i][j] = true;
                    else
                        dp[i][j] = dp[i + 1][j - 1];
                    if (dp[i][j])
                        result++;
                }
            }
        }
        return result;
    }
};
```
