# 718. 最长重复子数组

## 思路
>动态规划   

1. dp数组的含义  
dp[i][j]: nums1数组以i-1结尾，nums2数组以j-1结尾的最长重复子序列
2. dp数组的初始化   
为了递推的顺利进行，初始化dp[i][0] = 0, dp[0][j] = 0
3. 状态转移方程   
dp[i][j]可以由dp[i-1][j-1]推出   
如果`nums[i - 1] == nums[j - 1]` 那么`dp[i][j] = dp[i - 1][j - 1] + 1`  
4. 遍历顺序    
i, j的地位是等价的   


### ***C++***
```CPP
class Solution {
public:
    int findLength(vector<int>& nums1, vector<int>& nums2) {
        int size1 = nums1.size(), size2 = nums2.size();
        //dp[i][j]:A数组以下标i-1结尾，B数组以下标j-1结尾的最长重复子数组
        int result = 0;
        vector<vector<int> > dp(size1 + 1, vector<int>(size2 + 1, 0));
        int i, j;
        for (i = 1; i <= size1; ++i)
        {
            for (j = 1; j <= size2; ++j)
            {
                if (nums1[i - 1] == nums2[j - 1])
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                if (result < dp[i][j])
                    result = dp[i][j];
            }
        }
        return result;
    }
};
```
