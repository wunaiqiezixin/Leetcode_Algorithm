# 860. 柠檬水找零
[力扣题目链接](https://leetcode.cn/problems/lemonade-change/description/)  

---
## 思路
用一个数组记录当前手头上5， 10， 20美元的数量      
从左往右遍历数组   
如果遇到5美元，手头上5美元的数量+1  
如果遇到10美元，判断手头上是否有5美元的钞票，如果有5美元的钞票数量-1，10美元的钞票数量+1， 没有就返回false  
当遇到20美元时，优先考虑(10 + 5)的找零方式，再考虑 (5 + 5 + 5) 的找零方式   

### ***C++***
```CPP
class Solution {
public:
    bool lemonadeChange(vector<int>& bills) {
        int hash[3] = {0};//手头上5,10,20美元的钞票数量
        for (int i = 0; i < bills.size(); i++)
        {
            if (bills[i] == 5)
                hash[0]++;
            else if (bills[i] == 10)
            {
                if (hash[0] == 0)
                    return false;
                hash[0]--;
                hash[1]++;
            }
            else//要找15元的零，优先考虑10 + 5 ， 其次考虑 5 * 3
            {
                if (hash[0] > 0 && hash[1] > 0)
                {
                    hash[0]--;
                    hash[1]--;
                    hash[2]++;//可以删除，不用记录20美元的数量
                }
                else if (hash[0] >= 3 && hash[1] == 0)
                {
                    hash[0] -= 3;
                    hash[2]++;//可以删除，不用记录20美元的数量
                }
                else 
                    return false;
            }
        }
        return true;
    }
};
```
