# 1035. 不相交的线
[力扣题目链接](https://leetcode.cn/problems/uncrossed-lines/description/)   

---
本题本质就是[1143. 最长公共子序列](https://leetcode.cn/problems/longest-common-subsequence/description/)的换一个问法  

### ***C++***
```CPP
class Solution {
public:
    int maxUncrossedLines(vector<int>& nums1, vector<int>& nums2) {
        //即nums1、nums2的最大重复子序列长度
        vector<vector<int> > dp(nums1.size() + 1, vector<int>(nums2.size() + 1, 0));
        //dp[i][j]:
            // nums1[0, i - 1]、nums2[0, j - 1]的最大重复子序列长度
        int i, j;
        int result = 0;
        for (i = 1; i <= nums1.size(); ++i)
        {
            for (j = 1; j <= nums2.size(); ++j)
            {
                if (nums1[i - 1] == nums2[j - 1])
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                else
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
                if (dp[i][j] > result)
                    result = dp[i][j];
            }
        }
        return result;
    }
};
```
