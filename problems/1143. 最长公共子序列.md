# 1143. 最长公共子序列
[力扣题目链接](https://leetcode.cn/problems/longest-common-subsequence/description/)

---
## 思路
1. dp数组的含义  
`dp[i][j]`表示`nums1[0, i - 1]` 和 `nums2[0, j - 1]` 的最长公共子序列的长度  

2. dp数组的初始化   
`dp[0][j] = 0`, `dp[i][0] = 0`空串无公共子序列   

3. 递推公式    
如果`nums1[i - 1] == nums2[j - 1]`,   
那么`dp[i][j] = dp[i - 1][j - 1] + 1`  
如果`nums1[i - 1] != nums2[j - 1]`   
那么`dp[i][j]` = max(   
                    dp[i - 1][j],//nums1[0, i - 2] 与 nums2[0, j - 1]进行匹配    
                    dp[i][j - 1]// nums1[0, i - 1] 与 nums2[0, j - 2]进行匹配   
   
)

4. 遍历顺序：   
i - 1 -> i   
j - 1 -> j   
i和j地位一致   

### ***C++***
```CPP
class Solution {
public:
    int longestCommonSubsequence(string text1, string text2) {
        int i, j;
        int result = 0;
        //dp[i][j]:
        //  text1[0, i - 1], text2[0, j - 1]
        //  的最长公共子序列长度
        vector<vector<int> > dp(text1.size() + 1, vector<int>(text2.size() + 1, 0));
        
        for (i = 1; i <= text1.size(); ++i)
        {
            for (j = 1; j <= text2.size(); ++j)
            {
                if (text1[i - 1] == text2[j - 1])
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                else
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
                if (result < dp[i][j])
                    result = dp[i][j];
            }
        }
        return result;
    }
};
