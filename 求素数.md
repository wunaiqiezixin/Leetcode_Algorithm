# 如何求10000以内的素数

求10000以内的素数，并按 2，3，5，7 的格式打印

## 筛法求素数

* 用空间换时间
* 初始化一个_Bool类型的长度为10010的数组，首先0表示不是素数，1表示是素数
* 开始将下标2到10000的值均设为1
* 从下标2开始遍历数组，每遇到1，则将其下标的倍数均设为0

### ***C***
```C
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#define max_num 10000
_Bool isPrime[max_num + 10];

int main(void)
{
	for (int i = 2; i <= max_num; i++)
	{
		isPrime[i] = 1;
	}
	for (int i = 2; i <= max_num; i++)
	{
		if (isPrime[i])
		{
			for (int j = 2*i; j <= max_num; j += i)
			{
				isPrime[j] = 0;
			}
		}
	}

	printf("%d", 2);
	for (int i = 3; i <= max_num; i++)
	{
		
		if (isPrime[i])
		{
			printf(",%d", i);
		}
	}
 }
```
